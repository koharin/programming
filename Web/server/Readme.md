## webserver setting
```bash
git clone https://github.com/ungmo2/webserver-express.git
cd webserver-express
npm i // install express
```

## make working directory
```bash 
$ cd webserver-express
$ mkdir public
$ cd public
```
- `public` 디렉터리에서 작업한다.

## start webserver
```bash
npm start
```
- `http://localhost:3000/<filename>` 으로 접속한다.
